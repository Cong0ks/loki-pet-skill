# Loki 使用教程(新手友好版)

这份教程假设你完全没用过本项目,从零开始,每个功能都配"什么时候用 + 怎么操作"。
安装部分见 [INSTALL.md](INSTALL.md),这里只讲"装好之后怎么玩"。

---

## 1. 认识你的桌宠

启动(`python pet.py`,或作为 skill 时对 agent 说"启动桌宠")后,屏幕右下角会出现
一只漂浮的小幽灵 Loki。它的全部交互只有三个动作:

| 动作 | 效果 |
|---|---|
| **左键按住拖动** | 把它挪到任何位置,松手后原地继续漂浮 |
| **左键单击** | 打开/关闭聊天框 |
| **右键** | 打开菜单(所有开关都在这) |

聊天框右下角可以**拖拽调整大小**,调好后它会记住。

---

## 2. 和它聊天

单击 Loki → 输入框打字 → 回车。它会按人设(坏笑小幽灵)回复,并**开口朗读**。

- **用的是谁的额度?** 默认走你电脑上已登录的 agent CLI(如 Claude Code),
  用便宜的 Sonnet 模型,不用配任何 API key
- **嫌贵/想换模型?** 右键 → 聊天模型,选 Haiku(最省)
- **想完全独立的便宜接口?** 右键 → 对话后端 → 自定义 API,然后在聊天框输入:
  ```
  /api 你的key
  ```
  只给 key 默认用 DeepSeek;其他 OpenAI 兼容接口(Kimi/Qwen/中转站)写全三段:
  `/api 你的key https://接口地址/v1 模型名`
- **语音**:右键菜单可开关;换音色改 `config.json` 的 `tts_voice`
  (如 `zh-CN-XiaoxiaoNeural` 温柔女声)

> 聊天框里以 `/` 开头的是本地命令,不会发给 AI、不花钱。完整命令速查见文末。

---

## 3. 给它换动态表情(表情工坊)

Loki 可以播放透明背景的动画形象,素材来自**你自己的视频**(AI 生成的角色视频最合适,
要求纯色背景、角色有描边轮廓)。

1. 右键 → **表情工坊…**(或聊天框输入 `/face`)
2. 点"选择视频…"挑一个 mp4,起个名字,点"转换并导入"
3. 转换完成自动生效;面板里可以预览、切换、删除、恢复静态图片
4. 导入多个表情后,勾选面板左下角**"随机轮播"**,Loki 每隔几分钟自动换装

仓库自带两个示例视频(`assets/loki-DT1.mp4`、`loki-DT2.mp4`),可以直接拿来练手。

---

## 4. 让它帮你按 Yes(Claude Code 授权联动)

**痛点**:Claude Code 干活时频繁弹"是否允许执行?",人必须守在终端。

配好 hooks 后(见 INSTALL.md 第八节),每次要授权时:

1. Loki 弹出聊天框 + 语音喊你,显示要执行的命令
2. 🔎 几秒后出现**风险注解**——便宜模型用人话告诉你这条命令干什么、
   风险等级[低/中/高],看不懂命令也能决策
3. 点按钮:**允许**(放行一次) / **Yes 15分钟**(15 分钟内全部自动放行) /
   **拒绝** / **忽略**(立即回终端自己处理)
4. 5 分钟不操作会静默回落到终端,宠物端按钮自动消失,不会留死按钮

**临时授权**也可以随时右键开关("自动帮按 Yes")。
⚠️ 它会自动批准**所有**命令(包括删文件),只在自己盯着任务时用。

**任务完成通知**:跑超过 2 分钟的长任务结束时,Loki 弹泡泡 + 播放 BGM 报喜
(快问快答不打扰)。BGM 想换成自己的音乐,见 `assets/sounds/README.md`。

---

## 5. 人不在电脑前?(离开模式 + 邮件审批)

**痛点**:任务跑一半要授权,而你出门了。

**前置(一次性)**:安装腾讯 Agent Mail CLI 并 OAuth 登录(见 INSTALL.md 可选组件),
不需要任何密钥。

**使用**:出门前右键 → **离开模式(邮件审批)**。首次开启会弹框让你填常用邮箱
(填一次就记住,之后 `/email 新地址` 可改)。开启后:

- 授权请求自动发邮件到你手机,邮件里带命令详情 + 风险注解
- **直接回复邮件**,正文第一行写:`yes`(允许) / `no`(拒绝) / `15min`(临时授权)
- Loki 每分钟查一次回信,读到就放行,任务继续跑;授权最长等你 15 分钟
- 任务完成也会发邮件告诉你

回到电脑前记得右键关掉离开模式(每次重启宠物也会自动回到关闭状态)。

---

## 6. 额度跑满了?(计划任务)

**痛点**:订阅额度用完,Claude 提示"15:00 重置",你不想到点再手动敲"继续"。

聊天框输入:

```
/task 15:00 继续执行之前的任务
```

到 15:00,Loki 会自动在你最近那个项目目录里执行 `claude --continue`,
恢复最近的对话继续干活,完成后报喜(离开模式下发邮件)。配合"临时授权"或
"邮件审批",整个过程可以完全无人值守。

其他玩法:

| 命令 | 效果 |
|---|---|
| `/task +2h` | 2 小时后续跑(相对时间,分钟用 +30m) |
| `/task 09:00 提醒 开晨会` | "提醒"开头 = 只通知你,不动 Claude |
| `/task` | 查看所有待办任务 |
| `/task del 1` | 删除第 1 条 |

**两个重要注意**:

1. 续跑恢复的是**该目录最近的会话**,登记任务后别在同一目录再开新会话
2. 续跑是把整个会话重新加载后台执行——**会话越长,启动越慢、消耗额度越多**
   (几分钟到几十分钟都正常,宠物会一直等它并在完成时通知)。所以它最适合
   "中等长度的干活会话",不适合恢复聊了一整天的超长会话

## 7. 它会记住你(轻量记忆)

**痛点**:每次重启,Loki 都不记得你是谁、在忙什么。

开着就行,零操作:每聊 10 轮,Loki 会用便宜模型悄悄提炼"关于你的稳定事实"
(名字、偏好、在做的项目)和一句近况,存进 `~/.loki-pet/memory.md`(纯文本,
随时打开看/改),之后每次聊天自动带上——它就"认识你"了。

- 上限 40 条事实,每条带日期,满了自动淘汰最旧的;临时状态(今天很困)不会入库
- `/memory` 查看它记住了什么;`/memory forget 关键词` 定点删除;`/memory clear` 全清
- 不想要:config 里 `memory_enabled: false`
- 记忆只注入宠物自己的聊天,**不占用**你 Claude Code 工作会话的任何上下文

## 8. 命令速查表

| 命令 | 作用 |
|---|---|
| `/task 时间 [指令]` | 计划任务(续跑/提醒) |
| `/face` | 打开表情工坊 |
| `/email [新地址]` | 查看/修改离开模式通知邮箱 |
| `/api [key] [地址] [模型]` | 查看/配置自定义 API 后端 |
| `/memory` | 查看记忆 / `forget 关键词` 删除 / `clear` 清空 |

## 9. 右键菜单速查

| 菜单项 | 作用 |
|---|---|
| 表情工坊… | 导入/切换/轮播动态表情 |
| 自动帮按 Yes(15分钟) | 临时授权开关 |
| 离开模式(邮件审批) | 授权转发邮件开关 |
| 对话后端 | 宿主 Agent / 自定义 API |
| 聊天模型 | Haiku / Sonnet / Opus / 跟随宿主 |
| 语音朗读 | TTS 开关 |
| 退出 | 关闭宠物 |

## 10. 出问题了?

- 聊天报错 → 检查 `claude` 命令可用且已登录(或 `/api` 配置是否正确)
- 弹窗看不见 → 确认是用 `pythonw pet.py`(Windows)启动的
- 更多排查 → [INSTALL.md](INSTALL.md) 常见问题一节;程序异常日志在
  `~/.loki-pet/pet_error.log`
